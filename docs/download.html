Binaries

The binary versions of winLAME run on the following operating systems: Windows 7, Windows Vista, Windows 8, Windows 8.1 and Windows 10. The binaries may also run on Linux using Wine.

The latest build version of winLAME can always be found on AppVeyor: https://ci.appveyor.com/project/vividos/winlame/build/artifacts . The builds are run every time source code changes are committed and may by newer than the currently released version.

Setup package (recommended!)

The setup is provided as a Windows Installer .msi file and can be installed by double-clicking the downloaded file.

winLAME-Setup-2.21.1.108.msi (4.9 MB)
Download from Github.com

Standalone zip archive

The zip archive contains all files needed to run winLAME without installing it.

winLAME-2.21.1.108.zip (17.0 MB)
Download from Github.com

Note: There is no Portable Edition for winLAME this time, because the Portable package can't be built on command line using AppVeyor. Please use the Standalone zip archive instead.
