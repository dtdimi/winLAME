winLAME Classic Start Page

Classic UI only

This is the start page showing when winLAME is switched to the Classic UI mode or when restarting winLAME after a switch.

Buttons

Encode files

Select this when you want to encode files on your hard disc. Continues to the Input Files page.

Read CD

Select this when you want to read in an audio CD using a CD drive. Continues to the Input CD page.
