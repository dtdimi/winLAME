winLAME Wizard Pages

winLAME supports working in two different User Interface modes. You can always change between the modes, either by the icon on the ribbon in Modern UI mode, or in the system menu of the wizard pages in Classic UI mode.

Modern UI

In this mode the main window of the application shows all currently running encoding tasks, and new encoding tasks can be added instantly. Encoding tasks are set up using wizard pages, as in the Classic UI mode.

Classic UI

In this mode the wizard that you know from previous versions of winLAME is used. The wizard pages let you configure all details of encoding, and on the final encode page the current encoding task is completed.

The following configuration wizard pages can appear in winLAME. Some of the pages are only available in Classic UI or in Modern UI mode.

  • Classic Start Page (Classic UI only)
  • Input Files
  • Input CD
  • Output Settings
  • Preset Selection
  • LAME Settings
  • Opus Settings
  • Ogg Vorbis Settings
  • LibSndfile Settings
  • Windows Media Audio Settings
  • AAC Settings
  • Finish
  • Encode (Classic UI only)

The following settings pages can be shown in winLAME.

  • General Settings
  • CD Read Settings

Note

In winLAME, press F1 to get directly to the help page of the current wizard page. There are tool tips for every control on each page that show up when you hover over a control longer than one second.
