General Settings

This is the general settings page. The page can be shown when clicking on the "Settings" button in the Modern UI or when clicking on the "Options" menu entry in the system menu in Classic UI.

User Interface

Language

You can adjust the currently used language here. Currently supported are english and german. The new language is used after re-starting the application.

Number of CPU cores

This setting determines the number of CPU cores (or more exactly, execution threads) are used for encoding. The value can be chosen between 1 and 64, and the setting will be active after re-starting the application.

Automatically choose optimal number of CPU cores

This setting automatically chooses the number of CPU cores that are best for the current CPU. The number of cores actually used is shwon in brackets after the text. This check is set by default.
