Output Settings Page

This is the output settings page that configures general output settings.

Output Module

Lets you select the output module that winLAME should use to encode all of your files.

Warn about lossy transcoding

When this checkbox is active, a warning is shown when transcoding from one lossy format to another. This is done to accidentally prevent you to encode and lose quality. In some cases, transcoding is wanted, though.

Output Location

Lets you specify the output folder. The button at the left lets you select the folder. The folder history is available by clicking the down arrow.

Use input file's folder as output location

Uses the same output location as the input file has, for each input file. This allows multi-folder encodes. Note that this option is disabled when reading audio tracks from CD.

Delete after encoding

Indicates if the input file should be deleted after successful encoding of that file. Only use this if you don't need the input file anymore or when you have another copy of the input files. Note that this option is disabled when reading audio tracks from CD.

Overwrite existing files

Indicates if already existing files will be overwritten.

Warning

Be careful with this option when encoding to the sample folder as the input folder, or when delete files after encoding!

Create output playlist

When checked, an .m3u playlist with the name in the edit field is created while encoding files. Automatically enabled when a playlist was added or dropped into the input files list, or when a CD is read in.
