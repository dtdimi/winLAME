Opus Settings Page

This is the settings page for generating Opus audio files.

Bitrate

The bitrate setting determines the quality by using more bits per encoding frame, yielding in higher quality. Lower bitrates may disable some Opus algorithm details. Generally, Opus at 128 kbit/s is pretty much transparent. Command line option --bitrate

Complexity

This configures the the maximum algorithmic complexity. The default value of 10 means, that the encoder doesn't hesitate to use the CPU to give you the best quality encoding at the given bitrate. The allowed values are between 10 (highest CPU usage and quality) and 0 (lowest CPU usage and quality). You should always use the highest value of 10. Command line option --comp

Bitrate control mode

Controls how bitrate is maintained throughout the file.

VBR (recommended)

Uses variable bitrate encoding (default); command line option --vbr

Constrained VBR

Uses constrained variable bitrate encoding; command line option --cvbr

Hard CBR

Uses hard constant bitrate encoding; command line option --hard-cbr
