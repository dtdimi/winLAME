AAC Settings Page

This is the settings page for the AAC output module. AAC encoding and decoding is only available if the files "libfaac_dll.dll" (for encoding) and "libfaad2.dll" (for decoding) are present in the winLAME folder. These files are only available as source code, which must be compiled before using. For more infos, look at: https://github.com/knik0/faac.

Output Format

Quality

Sets default variable bitrate (VBR) quantizer quality in percent (default: 100, averages at approx. 120 kbps VBR for a normal stereo input file with 16 bit and 44.1 kHz sample rate. Max. value 500, min. 10.

Average Bitrate

Specifies the bitrate per channel the AAC encoder uses, in kbps. Although there are some fixed settings you can cycle through with the up-down-buttons, any bitrate can be typed into the field.

Allow Mid/Side

Allows the AAC encoder to use Mid/Side encoding. Stereo signals are separated in a mid (left minus right) channel, and a side (left plus right) channel. Should only be used when there is low stereo separation.

Use TNS

Tells the AAC encoder to use Temporal Noise Shaping.

Use LFE channel

Tells the AAC encoder to use one channel as LFE channel. libfaac doesn't currently support this.

Cutoff

Specifies the cutoff frequency the encoder uses of the audio input file, in Hz. Higher frequencies will be cut off.

MPEG Version

Lets you choose the MPEG version the output AAC file will be. Either MPEG2 or MPEG4. Note that this option doesn't create .mp4 files!

AAC Object Type

Lets you choose the object type that will be used for the output file.

Warning

libfaac currently doesn't support MPEG2 and the LTP object type at the same time.
